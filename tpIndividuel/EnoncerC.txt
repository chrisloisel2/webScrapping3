Objectif du TP:

Développer un script de web scraping qui navigue sur le site "Books to Scrape" et extrait les informations détaillées de chaque livre disponible.
Le script doit être capable de parcourir toutes les pages du site et de récupérer les informations suivantes pour chaque livre :

    Titre du livre
    Prix
    Disponibilité en stock
    Catégorie
    Note (si disponible)
    Description du produit (si disponible)
    URL de l'image de couverture

Sauvegarder les données extraites dans un fichier CSV séparé par catégorie.
 Vous devrez mettre en place une pipeline de traitement pour nettoyer et normaliser les données.
 Une seconde pipeline viendra enoyer les données nettoyées dans une base de données MongoDB.

 Exercice bonus : Vous leverez la main si vous avez fini l'exercice pour obtenir un exercice bonus.
